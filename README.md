# SH-VID2GIF

Convert videos to gifs.

## Prerequisites

- [ffmpeg](https://ffmpeg.org/)

## Installing

    make install

## Help

vid2gif

    Usage: vid2gif [-v][-o OFILE][-s SSEC][-l SDUR] [--] IFILE
    
    Convert input video files to GIF format.
    
    -l      : Limit the duration.
    -s      : Seek seconds before.
    -f RATE : Framerate (Default 10).
